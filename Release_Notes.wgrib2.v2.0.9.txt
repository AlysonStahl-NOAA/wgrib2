Implementation Instructions on WCOSS
------------------------------------------------
 - Checkout from git repository: 
    $ git clone https://github.com/NOAA-EMC/NCEPLIBS-wgrib2.git
    $ cd NCEPLIBS-wgrib2
    $ git checkout tags/wgrib2_v2.0.9
 - After building, the library, include files and executable
   will be placed in ./build/install
 - On Dell:
    - Invoke the build script as follows:
      $ ./build_all.sh dell
    - Move library, include files and executable to final location.
 - On Cray:
    - Invoke the build script as follows:
      $ ./build_all.sh cray 
    - Move library, include files and executable to final location.
